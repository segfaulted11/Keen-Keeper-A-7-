# Keen Keeper 

# Short Description
### KeenKeeper is a friendship management web application that helps users keep track of their relationships and interactions with friends. Users can view friend profiles, log interactions such as calls, texts, and video calls, monitor activity through a timeline, and visualize interaction patterns using analytics.

# Technologies Used
### Next.js (App Router)
### React
### Tailwind CSS
### DaisyUI
### Recharts
### React Context API
### Local Storage

# Key Features
### 1. Friend Management
### Browse a collection of friend profiles and view detailed information about each friend, including contact details, relationship goals, and interaction status.

### 2. Activity Timeline
### Record interactions such as calls, texts, and video calls with friends. Activities are displayed in a timeline and can be filtered by interaction type.

### 3. Friendship Analytics
### Visualize interaction data through charts and statistics. The analytics dashboard provides insights into communication patterns based on calls, texts, and video calls.
